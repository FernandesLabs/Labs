# Issue Template

**Report bugs, gaps, and improvements.**

## Problem

Clear description of what's wrong or missing.

**Summary**: One-line summary of the issue.

## Expected

What should happen? Be specific.

## Actual

What actually happens? Include error messages.

## Reproduction

Steps to reproduce:

1. Go to [page]
2. Click [button]
3. See [error]

## Environment

- Browser: [version]
- OS: [platform]
- Product: [name]

## Severity

- **Critical**: Breaks core functionality
- **High**: Affects many users
- **Medium**: Affects some users
- **Low**: Minor improvement

## Proposed Solution

If you have ideas for how to fix this, describe them.
